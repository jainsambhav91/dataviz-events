Weekly Assignment 8

Due 11/9 at 4:00pm

Continue working with the sensor you chose in class.

Decide how you will structure data from your sensors into a table (or multiple tables) in a SQL database. Use these decisions to define a data model/schema for your table(s). Be very specific about the data types you will use for each column. Use this template, which shows a fictitious example with two tables. One table collects sensor values and the date and time the sensor value is collected. The other table contains weather observations and the date the observation was collected. The tables could be joined on the date column.

Think about how to collect the maximum amount of data with the minimum data storage footprint. Your data model/schema should plan for a normalized relational database.
